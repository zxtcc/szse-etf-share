## 五、已知限制与说明
- 数据依赖深交所公开接口，需联网访问；接口结构变化时需同步调整 `szse_client.py`。
- 因深交所证书链不在 Python 默认 CA bundle，请求使用 `verify=False`（已抑制告警）。
- 批量回填逐日请求并限速（每次 0.3s），大区间耗时较长，已限制单次 ≤400 天。
